# **App Name**: Daily Four

## Core Features:

- Daily Journal Navigation: A home screen calendar/list allows users to easily view and access daily entries. Quick access to 'Today' and browsing past days.
- Automated Daily Pages: Each day automatically generates four dedicated pages: a customizable checklist, two reflection sections, and a mood/free-writing page. Users never manually create pages.
- Configurable Daily Checklist: An interactive checklist with tappable items and a progress indicator (e.g., 'X/Y completed'). Checked states are preserved for each specific date.
- Guided Reflection Pages: Dedicated writing sections for 'What I am grateful for,' 'What I learned today,' 'What drained me of energy,' and 'What to improve,' all with automatic saving while typing.
- Mood Tracking & Open Journal: Allows users to select their 'Mood of the day' using emojis or predefined buttons, followed by a large, uncluttered area for free-form journaling with autosave.
- Local-First Data Storage: All journal entries, checklist states, and mood selections are securely saved and persisted locally on the device, ensuring privacy and quick access.
- Reflection Prompt Tool: An AI tool that gently suggests personalized writing prompts or provides nudges within empty reflection sections, designed to inspire thought and encourage daily journaling.

## Style Guidelines:

- Primary color: A gentle, warm peach (#E6D8CE) evoking softness and calm. This color anchors the user interface without being overly dominant.
- Background color: A very light, subtly warm off-white (#FCFAFA), ensuring a peaceful, uncluttered canvas that feels warm and inviting.
- Accent color: A soft, muted rose (#FFD6D6) for interactive elements and gentle highlights, adding a touch of feminine warmth and visual interest.
- Headlines will use 'Belleza' (humanist sans-serif) for an artistic and refined touch, while body text will use 'Alegreya' (humanist serif) for elegant readability in longer entries.
- Utilize delicate, minimal line-based icons for navigation and functional elements, maintaining a clean and unobtrusive aesthetic. Mood selection will be supported by a curated set of soft-colored emojis or custom, expressive illustrations.
- A clean, mobile-first layout organized around generously spaced, rounded-corner cards for all content, enhancing the soft and accessible feel. Emphasize ample white space and clear visual hierarchy for peaceful readability and ease of use, especially in low light.
- Implement gentle, fluid swipe transitions between the four daily pages to provide a smooth, tactile navigation experience. Subtle fades and scale effects will be used for interactive elements and state changes, contributing to a calm and premium feel.
